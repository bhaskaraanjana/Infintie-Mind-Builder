# Keyboard Shortcuts

## General
| Shortcut | Action |
| --- | --- |
| `Cmd` + `K` / `Ctrl` + `K` | Open Global Search |
| `Esc` | Close Modals / Search |

## Canvas Actions & Navigation
| Shortcut | Action |
| --- | --- |
| `Delete` / `Backspace` | Delete Selected Notes |
| `Wheel` | Panning (Vertical) |
| `Cmd` + `Wheel` / `Ctrl` + `Wheel` | Zoom In/Out |
| `Pinch` (Trackpad) | Zoom In/Out |
| `Space` + `Drag` | Pan Canvas (Mouse) |

## Note Editor
| Shortcut | Action |
| --- | --- |
| `Cmd` + `Enter` / `Ctrl` + `Enter` | Save & Close Editor (if applicable) |
| `Esc` | Close Editor (if focused outside text) |

## Text Formatting (Rich Text)
| Shortcut | Action |
| --- | --- |
| `Cmd` + `B` / `Ctrl` + `B` | **Bold** |
| `Cmd` + `I` / `Ctrl` + `I` | *Italic* |
| `Cmd` + `Shift` + `X` / `Ctrl` + `Shift` + `X` | ~~Strikethrough~~ |
| `Cmd` + `E` / `Ctrl` + `E` | `Inline Code` |
| `Cmd` + `Alt` + `C` / `Ctrl` + `Alt` + `C` | ```Code Block``` |
| `Cmd` + `Shift` + `7` / `Ctrl` + `Shift` + `7` | Numbered List |
| `Cmd` + `Shift` + `8` / `Ctrl` + `Shift` + `8` | Bullet List |
| `Cmd` + `Shift` + `B` / `Ctrl` + `Shift` + `B` | Blockquote |

## Tagging
| Shortcut | Action |
| --- | --- |
| `Enter` | Add Tag |
| `,` (Comma) | Add Tag |
| `Backspace` | Remove last tag (if input empty) |

*(Note: Some shortcuts may vary depending on Operating System)*
